# 📝 To-Do API — ASP.NET Core (.NET 8)

Uma API REST simples para gerenciar tarefas (To‑Do), construída com **ASP.NET Core (.NET 8)** e **Entity Framework Core**.  
Este projeto foi criado para prática de Web API, CRUD, Controllers, Rotas, Injeção de Dependência e integração com banco de dados.

---

## 🚀 Tecnologias utilizadas

- **.NET 8**
- **ASP.NET Core Web API**
- **Entity Framework Core**
- **Swagger / OpenAPI**
- **PostgreSQL** (via connection string )
- **C#**

---

## 📌 Funcionalidades

- Criar uma tarefa (`POST`)
- Listar todas as tarefas (`GET`)
- Buscar uma tarefa por ID (`GET /{id}`)
- Atualizar tarefa (`PUT`)
- Excluir tarefa (`DELETE`)
- Integração com banco PostgreSQL (sem expor credenciais)
